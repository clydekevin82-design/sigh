# GameBanana Upload Kit

Upload the Windows release zip generated by GitHub Actions or:

```powershell
pwsh scripts/package-windows.ps1 -ReleaseVersion v0.1.0
```

Use the zip from:

```text
artifacts/LtdMusicImporter-v0.1.0-win-x64.zip
```

Do **not** upload:

- `RomFS/`
- `ExeFS/`
- `.nsp`, `.xci`, game updates, keys, or firmware files
- Generated mods that contain copyrighted game assets beyond user-created
  replacement files
